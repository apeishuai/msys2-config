Creating animations

Using Greenfish Icon Editor Pro, you can also create animated cursors and GIF files. All you have to do is to draw all frames as individual pages, then configure how long each of them should be shown. This is called the frame duration. When creating an animated GIF, you can also specify the loop count by choosing File | Metadata. Note that the loop count is not saved when using another file format.

To create a new frame, just select a page in the list, then press New Page to set frame dimensions and duration. The new frame will be based on the page previously selected.

To set the same duration for all frames:

  • Double-click a page and set the desired duration for the corresponding frame in the dialog box.

  • Keep this page selected and choose Icon | Apply frame duration to all frames.

You can test your animation like testing an icon or cursor, choosing Icon | Test or clicking on the magnifying glass icon on the document toolbar.
