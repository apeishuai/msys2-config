Creating a new graphic

To create a new graphic, use the File | New graphic menu item, or double-click on the gray main window and choose New graphic. You will be asked to enter the properties of the page (icon format, frame) which is about to be created. You cannot select the color depth, because it will be dynamically determined later.

Tip: When creating icons, you should start with creating and drawing a 256x256 image format, because using your icon under Windows Vista(tm), it is recommended to have an image format in such a large size. After that, you will be able to create all other formats easily, using the Icon | Create icon from image menu item.
