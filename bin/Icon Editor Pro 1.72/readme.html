  • Delete the program folder
Version history

  • 1.72

    • Added feature: allowing zoom ratios smaller than 1
    • Added feature: hiding/showing list of pages

  • 1.71

    • Fixed some minor bugs
    • Added feature: setting max. image width and height separately

  • 1.7

    • Added feature: opening, editing, saving executable/resource files, using an integrated Resource Editor
    • Added Russian help (can be downloaded from the Greenfish Homepage)
    • Changed: "Transform" dialog box: you don't have to set Units separately for Width and Height
    • Added language packs: Greek, Ukrainian

  • 1.6

    • Fixed bug: when copying to clipboard, inverted parts were replaced with the transparent color
    • Added feature: GIF support
    • Added feature: assembling ICL (icon library) and RES (Win32 resource) files
    • Added feature: editing Metadata (ANI summary info, JPEG quality, GIF loop count)
    • Added feature: the canvas can also be enlarged using the Crop tool
    • Added feature: exporting individual pages
    • Added language packs: Chinese (traditional), Dutch, Russian

  • 1.51

    • Fixed bug: a few virus scanners reported .ANI files created by GFIE as "Suspicious", now GFIE creates .ANI files considered "Clean" by them
    • Fixed bug: opening odd (uneven)-sized icons with GFIE, the transparency became a mess
    • Added feature: menu item for cropping transparent parts
    • Added feature: using custom background images in the Test dialog box
    • Added feature: using the + - * / shortcuts for zooming
    • Added feature: the mouse wheel can be set for scrolling or zooming under Help|Preferences
    • Added feature: specifying a target folder for batch converted files
    • Added language packs: Norwegian, Portuguese (Brazilian), Serbian (Latin)

  • 1.5

    • Moved Preferences... menu item to Help menu and added a separate Language... menu item
    • Fixed bug: viewport position was always reset to image center when setting the zoom in a document window
    • Fixed bug: some PNG files were not able to be opened
    • Fixed bug: opening a lot of (~30) files and using the hot spot tool greatly decreased performance
    • Fixed bug: selecting a format at the bottom of the list sometimes caused it to be moved
    • Fixed bug: pressing the right mouse button while drawing with the left caused strange results
    • Optimized: now creating better compressed PNG files
    • Added feature: working with images up to 1024x1024 (upper limit can be increased at your own risk)
    • Added feature: editing and testing animations, saving them in ANI (animated cursor) files
    • Added feature: customizing grid and center line colors
    • Added language packs: Korean, Polish, Spanish

  • 1.4

    • Fixed bug: compatibility issues: created icons were recognized by Windows, but not always by other graphics programs
    • Fixed bug: when changing language pack at Preferences, dialog view mode changed to erroneous value -1
    • Fixed bug: associated files were sometimes opened twice
    • Fixed bug: Bevel filter generated too crisp results at big radiuses
    • Fixed bug: custom colors set in windows color dialogs were lost between sessions
    • Added feature: batch converting image files
    • Added feature: Tool Behavior/Color Picker windows can be switched to Compact mode
    • Added feature: showing center lines of images
    • Added feature: pasting 32-bit images from graphics apps which know the image/png clipboard format, e.g. from GIMP
    • Added language packs: French, Chinese (simplified)
    • Minor corrections to other language packs

  • 1.31

    • Fixed bug: default file extension was not appended when using File|Save as
    • Added language pack: Italian
    • Minor corrections to other language packs

  • 1.3

    • Added feature: multilanguage support
    • Added feature: using Arrow keys to move view/selection
    • Added feature: greater values than 100 are also accepted at Filters|Opacity
    • Added feature: workspace layout (i.e. window positions) is autosaved
    • "Preferences" dialog was rearranged, new option for setting view mode in Open/Save dialog boxes

  • 1.2

    • Added feature: drawing gradients in color/transparency mode (for creating fadeouts)
    • Added feature: startup screen
    • Minor GUI modifications

  • 1.1

    • Rearranged the "Tool Behavior" dialog box
    • Fixed bug: fake tool button at the bottom of the toolbar window
    • Added feature: file drag and drop support
    • Added feature: new brush shapes (horizontal, vertical, spray)
    • Added feature: now the strength of the Eraser tool can be adjusted from 1% to 100%
    • Added feature: using "Thumbnails" mode also when opening selections/exe files
